01

El Problema

Muchas personas tienen dificultades para gestionar sus tareas diarias.

En la actualidad, tanto en el ámbito académico como personal, la mayoría de las personas administran sus actividades de forma manual o con herramientas poco estructuradas como notas físicas, aplicaciones básicas o simples recordatorios mentales.

Esto provoca retrasos, olvidos y una disminución importante en la productividad. Además, las herramientas actuales no suelen integrar todas las funciones necesarias en un solo lugar.

  • ! Falta de organización en las actividades
  • ! Olvido frecuente de tareas importantes
  • ! Ausencia de priorización clara
  • ! Dificultad para visualizar el progreso
  • ! Uso de múltiples aplicaciones al mismo tiempo
02

Nuestra Solución

Una aplicación web que centraliza la gestión de tareas en un solo lugar.

Proponemos el desarrollo de una aplicación web basada en Componentes Web, que permite crear tareas con título, descripción y estado, organizarlas por categorías, recibir recordatorios, visualizar el rendimiento con gráficas y almacenar recursos útiles. Toda la información se guarda con LocalStorage, sin necesidad de un servidor externo.

📋

Organización clara

Las tareas se agrupan por estado: pendiente, en progreso y completado, para visualizar fácilmente el progreso.

🔔

Recordatorios

Sistema de notificaciones que avisa sobre tareas importantes para evitar olvidos.

📊

Rendimiento

Gráficas dinámicas que muestran el avance del usuario según las tareas completadas y pendientes.

Objetivo General

🎯

Desarrollar un Sistema Inteligente de Gestión de Tareas basado en Componentes Web, que permita a los usuarios organizar, gestionar y dar seguimiento a sus actividades de manera eficiente, integrando funcionalidades como recordatorios, visualización del rendimiento mediante gráficas y almacenamiento de recursos, con el fin de mejorar la productividad y optimizar la administración del tiempo.

06

MVP del Sistema

Producto Mínimo Viable: las funciones esenciales de la primera versión.

Incluido En esta versión

  • Creación de tareas con título y descripción
  • Visualización de la lista de tareas
  • Edición y eliminación de tareas
  • Clasificación por estado (pendiente / en progreso / completado)
  • Almacenamiento de datos mediante LocalStorage
  • Interfaz básica, intuitiva y funcional

Fuera Para futuras versiones

  • Sistema avanzado de notificaciones automatizadas
  • Visualización de gráficas de rendimiento
  • Repositorio de recursos
  • Integración con bases de datos externas o en la nube
  • Sistema de autenticación de usuarios
  • Aplicación móvil nativa
